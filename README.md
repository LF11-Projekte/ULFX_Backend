# Environmentvariables

- BACKEND_URL
  - URL des Backends
  - DEFAULT: http://localhost:8000
- FRONTEND_URL
  - URL des Frontends
  - DEFAULT: http://localhost:5173
- USERMANAGER_URL
  - URL des Usermanagers
  - DEFAULT: http://localhost:3000
